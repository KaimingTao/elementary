# Elementary

Version: [2025-06-17](https://kaimingtao.github.io/Elementary/)

When we live, we think, but the world is not simple, is it possible to make it easier? Then the question is about **How to**. I find some methods which are basic and general, using them I can create useful tools which solve life questions, or open new ways of seeing things. I'd like to share these knowledge.

Although these methods are considered general, I'd prefer to call them life hacking method or life tricks, I cannot test them on all possibilities. I'm open to find new ways if these methods don't work.

I'm trying to make this project concise, and also trying to illustrate the methods in different ways which helps me see things in different angles and trains my skill of communication.

My goals is to provide a guide without confusion because *Life* are complicated, I just want to make it a bit understandable, controllable, and make it a bit easier, also have some fun.

## Communication is the basic skill

Communication is the basic skill for any kinds of work, make a clear plan, self communication, communicate with others, design a complex system, etc. That's because an entity no matter how small like an atom or large like a world is build by components, then coordinating by switching information, hence communication.

Expression is not communication, expression may not consider if other people can understand, but communication should. But they're related.

## Split complex to simple ones

Because any kind of complex object, no matter how large or how small, they're built from lower level items or concepts, so when we cannot handle something, we can try to catch parts of it, build a mind model piece by piece, finally we can grasp some tricks or general methods. Also the process of breaking complex to simple, then rebuild from bottom to top is a natural way to be better.
